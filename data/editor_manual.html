EDITING · BASICS
기본 편집 — 선택 · 이동 · 리사이즈 · 텍스트
중앙 캔버스의 요소를 클릭해 선택하면 우측 패널에 속성이 나타납니다.
🖱 선택 & 이동
1

요소 클릭

마우스 오버 시 점선, 클릭 시 초록 핸들로 변경됩니다.

2

드래그로 이동

선택된 요소 위에서 드래그하면 자동으로 position:absolute로 전환됩니다.

3

핸들로 리사이즈

모서리·변의 8개 핸들을 끌어 크기 조절. 비율 유지는 모서리 핸들 사용.

⌨
키보드 단축키

DEL 삭제 · Ctrl+D 복제 · ESC 선택 해제 · 방향키 1px nudge

✏ 텍스트 편집

요소를 더블클릭하면 인라인 편집 모드로 들어갑니다. Enter로 줄바꿈, Esc로 종료. 우측 패널의 텍스트 영역에서 HTML 모드로 한 번에 수정할 수도 있습니다.

🎨 우측 속성 패널

위치(left/top), 크기(W/H), 폰트, 색상, line-height, letter-spacing 등을 직접 입력. 적용 범위 셀렉터로 자식 요소까지 강제 덮어쓰기 가능.

💡
선택 우선순위

슬라이드 배경을 클릭하면 .slide 자체가 선택됩니다. 자식 요소를 선택하려면 그 요소 위에서 직접 클릭하세요.

좌측 패널

썸네일 클릭으로 슬라이드 이동. 휴지통 아이콘으로 삭제, 드래그로 순서 변경.

+

새 슬라이드 추가

상단 툴바의 + 슬라이드 버튼으로 빈 슬라이드 추가, 복제로 현재 슬라이드 통째 복사.

↩
실행 취소 / 재실행

Ctrl+Z 취소, Ctrl+Shift+Z 재실행. 모든 편집은 히스토리에 자동 기록됩니다.

💾 저장 & 불러오기

상단 툴바의 저장 버튼으로 단일 HTML 파일 다운로드. 파일명은 원본_edited.html 형식. 파일 열기로 다른 HTML PPT 불러와 편집 시작.

📄 새 PPT는 프롬프트 템플릿으로

툴바의 📄 프롬프트 템플릿 버튼을 눌러 .md 파일을 받은 뒤, ChatGPT/Claude에 붙여넣고 주제만 추가하면 편집기 호환 HTML이 생성됩니다.

⚠
주의

저장은 브라우저 다운로드로만 이뤄집니다. 자동 저장 없음 — 큰 변경 후에는 수시로 저장하세요.
