# HTML PPT Edit

브라우저에서 동작하는 단일 파일 HTML 슬라이드 편집기.

## 사용

- 라이브: https://muttul58-coder.github.io/HTML_PPT_EDIT/
- 로컬: `index.html`을 브라우저로 열기

## 구조

- `index.html` — 편집기 본체 (GitHub Pages 진입점)
- `editor.html` — 동일 파일 (호환용)
- `data/` — 샘플 슬라이드

## 배포

`main` 브랜치에 푸시하면 GitHub Pages가 자동 배포합니다.
저장소 → Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `(root)`.
